Interactive geometry, plotting, visualization

JSXGraph is a cross-browser JavaScript library for interactive geometry, function plotting, charting, and data visualization in the web browser.

Features

  • Euclidean and projective Geometry
  • Curve plotting
  • Open source
  • High-performance
  • Small footprint
  • No dependencies
  • Multi-touch support
  • Backward compatible down to IE 6

JSXGraph is implemented in pure JavaScript, does not rely on any other library, and uses SVG, canvas (or even the ancient VML). Special care has been taken to optimize the performance.
